# BFJS

[![Build Status](https://travis-ci.org/KevinGrandon/bfjs.svg?branch=master)](https://travis-ci.org/KevinGrandon/bfjs)

A brainfuck experiment in javascript, using node.js.

## Usage

Interpreter Usage

```
./interpret "++++++++[<++++++++>-]<+."
./interpret "++++++++[>++++[>++>+++>+++>+<<<<-]>+>+>->>+[<]<-]>>.>---.+++++++..+++.>>.<-.<.+++.------.--------.>>+.>++."
```


Parser Usage

```
./parse "++++++++[<++++++++>-]<+."
```


Genetic Algorithm Finder

```
./genetic
```
