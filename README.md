# BFJS

A brainfuck experiment in javascript, using node.js.

## Usage

Interpreter Usage

```
./interpret "++++++++[<++++++++>-]<+."
./interpret "++++++++[>++++[>++>+++>+++>+<<<<-]>+>+>->>+[<]<-]>>.>---.+++++++..+++.>>.<-.<.+++.------.--------.>>+.>++."
```


Parser Usage

```
./parse "++++++++[<++++++++>-]<+."
```


Genetic Algorithm Finder

```
./genetic
```
